Web-connected 3D Printer in garage (OctoPrint)

3D printing is rewarding and useful. Even more so when it can be done from anywhere in the world! Thanks to the open-source framework OctoPrint, all it takes is a Raspberry Pi and a little Linux knowledge start and monitor 3D prints from any web browser.

If my printer is currently online, tune in to see if something is in the works!

BoomBox

I really enjoy going to the shooting range and punching some holes in some paper. My roommate and I thought it'd be fun to design a device you could attach to a pistol to log motion data and begin extrapolating 'good' and 'bad' habits in order to help people improve and challenge their friends! While we only ended up getting as far as a very rough prototype, this spun off an idea which I continued to run with: a web application which automatically scores pictures of targets which you've shot, ShotLog.us!

ShotLog.us

It's currently up on the web, running on AWS EC2, AWS S3, Ruby on Rails, and OpenCV. Though it only has the core feature of 'scoring' the target. At some point I'll come back and implement the remaining core features: improvement tracking and friend leader-boards!

Feel free to give it a try! The standardized target to print and shoot is linked on the main page.

A special thanks to my friend mjsteger, for helping me glue my code together, and to Michael Hartl's Rails Tutorial, which got me going on RoR development!

Mechanical Keyboards

MicroMechBoard

I stumbled across an Open Source Hardware project of a small, split-hand mechanical keyboard. The project code and hardware files for Will Yager's MicroMechBoard are all excellent and posted on GitHub. I followed along with his instructions and blog post and the keyboard came out exactly as advertised!

ErgoDox

Sometime later, I built another split-hand mechanical keyboard called the ErgoDox for my friend (the one who helped me with ShotLog, in fact!), and it was a wonderful experience. All of this following along in the design of a keyboard made me bold enough to begin my own forray into it.

mGH60

Thanks to the generous and open mechanical keyboard community, I forked user komar007's GH60 keyboard design. I've started by shrinking the hardware onto a small, reprogrammable board which I can use to begin prototyping. I've named the design the mGH60, a 'miniature' version of the original. As with the original design, I've used KiCAD to modify the existing files.

I've shared the OSH Park project so you can download or fabricate the mGH60 too!

The next step will begin playing around in the tmk_keyboard code to get the mGH60 communicating with the BLE113 Bluetooth 4.0 module I already have. If all goes well, a "full-sized" 60% PCB design will be next!
